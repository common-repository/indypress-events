Stable tag: trunk

Indypress Events is a plugin which creates and manages a post_type suitable
for event, with start/end, location, SEO optimization, dedicated widget and
more.
It is now _completely_independent_ from Indypress plugin (you can install
it without installing indypress at all), but if you use both, they're well
integrated

== Description ==

### Features

* Add Event UI
* datetime start/end
* location
* Widget SEO-optimized using "calendar" microformat
* Customizable permalink structure (search engines will like this!)

#### Open Publish form features

If you use Indypress, there are special "inputs" for it
